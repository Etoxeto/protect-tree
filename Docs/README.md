# Docs Index

Start here before changing the project:

1. `Docs/CURRENT_STATE.md` - current source of truth, active priorities, and
   common traps.
2. `Docs/ARCHITECTURE.md` - authority boundaries, scene composition, board,
   networking, and Boss contracts.
3. `Docs/PROJECT_PLAN.md` - milestone history and current stage direction.
4. `Docs/GAME_DATA_GUIDE.md` - designer-facing data editing guide for pieces,
   enemies, synergies, waves, shop, timing, and catalogs.
5. `Docs/M4_LAN_MULTIPLAYER.md` - LAN lobby, match transport, reconnect, and
   snapshot rules.
6. `Docs/BOSS_WAVE_RULES.md` and `Docs/BOSS_SKILLS.md` - current Boss wave and
   Boss skill authority/presentation contracts.
7. `Docs/XLUA_STARTUP_EXPERIMENT.md` - XLua startup, generated bindings, and
   Windows/Android script-loading experiments.
8. `Docs/VFX_GUIDE.md` and `Docs/VFX_RUNTIME_SORTING.md` - visual effects and
   runtime sorting notes.

When a task changes gameplay rules, architecture boundaries, scene workflow,
network protocol, validation status, or development direction, update the
matching document in the same task.
